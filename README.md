# User & Order Management REST API

## It is backend project using Typescript to understand fundamental of REST API

This project is a REST API server designed to manage users, products, and orders. It demonstrates core backend concepts, including REST API design, database modeling, authentication/authorization, error handling, and deployment readlines.

Features:
*Authentication & User Management
*User registration (signup)
*Login functionality
*JWT-based authentication
\*Role-based access control (User / Admin)

Product Management:
*Admin-only product registration
*Product listing and retrieval
*Search and pagination support
*Order Management
*Create new orders
*View order history
\*Track order status: Created / Paid / Canceled

General Backend Features:
*Proper RESTful design
*Database modeling (MongoDB / MySQL)
*Global error handling
*Validation of input data
